Guessing number game

There is a guessing number game running in the developer console.

How to play?

  1. Open the browser's developer console, normally the shortcut is F12 or Ctrl + Shift + I or K
  2. Input your guess with a Javascript expression, like num = 50 or num = 20
  3. The number range is from 0 to 100
  4. There is a trick exposed if you really tried many times
